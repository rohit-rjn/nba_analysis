Logistic Regression for Win Predictibility¶

In this report I am trying to predict win probability of a team in a given match
To build a Good Logistic Regression Model, I am trying to build a good feature set
The feature set containes:
1) Home Team vs Visiter Team
2) Perfomrance by team in last season (Win/Losses)
3) Performance in this season till now (Win/Losses)
4) Points Used Till Mid-Time

The abvoe feature set is used, as it available pre-match, and we are trying to predict the win probability in two steps
1) Before the Start of Match based on first 3 stats,
2) After Half Time by using 2 QTRs data, and also transforming into mean Diff by first half

    df_team[column] = 0
for column in columns_factor:
    df_team[column] = 0
df_team['NO_GAMES'] = 0
for i in range(len(df_team)):
    df_sum = df[df.TEAM_ID == df_team['TEAM_ID'].iloc[i]].sum()
    df_mean_factor =  df_factor[df_factor.TEAM_ID == df_team['TEAM_ID'].iloc[i]].mean()
    for column in columns:
        df_team[column].iloc[i] = df_sum[column]
    for column in columns_factor:
        df_team[column].iloc[i] = df_mean_factor[column]
        
    df_team['NO_GAMES'].iloc[i] = df[df.TEAM_ID == df_team['TEAM_ID'].iloc[i]].GAME_ID.count()
